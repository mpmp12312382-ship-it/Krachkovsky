# Expense Tracker

**Автор:** Крачковский Илья Андреевич

## Описание программы

Expense Tracker - это графическое приложение для отслеживания личных расходов. Программа позволяет добавлять записи о расходах, фильтровать их по категории и дате, а также подсчитывать сумму расходов за выбранный период. Данные сохраняются в JSON-файл и автоматически загружаются при запуске.

## Требования

- Python 3.6 или выше
- Стандартные библиотеки: tkinter, json, os, datetime

## Установка и запуск

1. Скачайте файл `expense_tracker.py`
2. Запустите программу:
```bash
python expense_tracker.py
