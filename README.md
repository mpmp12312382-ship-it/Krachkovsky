# Movie Library

**Автор:** Крачковский Илья Андреевич

## Описание программы

приложение для хранения и управления информацией о фильмах.
озволяет добавлять фильмы, фильтровать, а также сохранять данные.

## Требования

- Python 3.6 или выше
- Стандартные библиотеки: tkinter, json, os

## Установка и запуск

1. Скачайте файл `movie_library.py`
2. Запустите программу:
```bash
python movie_library.py
